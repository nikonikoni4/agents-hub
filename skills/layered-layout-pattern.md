# 层叠布局设计模式 (Layered Layout Pattern)

## 核心理念

将界面分为 **底层** 和 **上层** 两个视觉层级，通过背景色差异、阴影和圆角营造空间纵深感，而非依赖线条分隔。

## 层级结构

```
┌─────────────────────────────────────────────┐
│  TopBar (底层背景色)                         │  ← 底层
├──────────┬──────────────────────────────────┤
│          │ ┌──────────────────────────────┐ │
│  左侧栏  │ │  主内容区 + 右侧栏           │ │  ← 上层（浮层）
│ (底层色) │ │  (上层背景色，带阴影+圆角)    │ │
│          │ └──────────────────────────────┘ │
└──────────┴──────────────────────────────────┘
```

## 两层定义

| 层级 | 组成 | 背景色 | 视觉特征 |
|------|------|--------|----------|
| **底层** | TopBar + 左侧栏 | 浅灰色（如 `#f7f7f8`） | 无阴影、无边框线、无缝融合 |
| **上层** | 主内容区 + 右侧栏 | 白色（如 `#fcfcfc`） | 阴影浮起、外侧圆角、形成整体卡片感 |

## CSS 实现要点

### 1. 容器（底层背景）

```css
.mainContainer {
  display: flex;
  flex: 1;
  overflow: hidden;
  /* 关键：左侧和顶部 0 padding 让底层紧贴边缘 */
  /* 右侧和底部留 padding 给上层阴影展示空间 */
  padding: 0 6px 6px 0;
  gap: 0;
  background: var(--bg-sidebar); /* 底层灰色 */
}
```

### 2. TopBar（与左侧栏同层）

```css
.topBar {
  height: 32px;
  background: var(--bg-sidebar); /* 与左侧栏同色，无缝融合 */
}
```

### 3. 左侧栏（底层，无装饰）

```css
.leftSidebar {
  width: 220px;
  flex-shrink: 0;
  background: var(--bg-sidebar); /* 底层色 */
  /* 无 border、无 box-shadow、无 border-radius */
}
```

### 4. 主内容区（上层浮层，左侧圆角）

```css
.chatArea {
  flex: 1;
  background: var(--bg-main); /* 上层白色 */
  border-radius: 12px 0 0 12px; /* 左侧圆角，右侧与右侧栏衔接 */
  box-shadow: -2px 0 8px rgba(0, 0, 0, 0.06); /* 向左投射阴影，暗示浮起 */
}
```

### 5. 右侧栏（上层浮层，右侧圆角）

```css
.rightSidebar {
  width: 220px;
  background: var(--bg-main); /* 与主内容区同色，形成整体 */
  border-radius: 0 12px 12px 0; /* 右侧圆角 */
  box-shadow: 2px 0 8px rgba(0, 0, 0, 0.06); /* 向右投射阴影 */
}
```

## 关键设计原则

1. **颜色分层，不用线条** — 两层用不同背景色区分，去掉所有 `border` 分隔线
2. **阴影暗示高度** — 上层用 `box-shadow` 浮起，阴影方向朝底层（左右外侧）
3. **圆角塑整体感** — 主内容区左圆角 + 右侧栏右圆角 = 合起来是一个完整的浮层卡片
4. **底层无缝融合** — TopBar 和左侧栏同色同层，无视觉断裂
5. **padding 方向** — 容器只在阴影外露方向留 padding（右侧和底部），其余方向紧贴

## 适用场景

- 工具型应用（IDE、协作平台、管理后台）
- 侧栏导航 + 主内容区的经典布局
- 需要层次感但不想用深色/亮色强对比的场景

## 反模式

- ❌ 用 `border` 线条分隔各区域（扁平、无层次）
- ❌ 所有面板同一背景色（失去空间感）
- ❌ 所有面板都有阴影（层级混乱，不知道谁在上）
- ❌ 内部子模块也有大阴影（与外层浮层竞争注意力，内部模块用轻阴影 `0 2px 6px` 即可）
