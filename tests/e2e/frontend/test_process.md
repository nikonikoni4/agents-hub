## 关键发现（调研阶段）
- 前端无 React Router，视图切换通过 `viewMode` 状态控制（chat/role/skill）
- 状态管理：Zustand，每个 feature 独立 store
- 后端 API 挂载在 `/api/v1` 前缀
- WebSocket 端点：`ws://localhost:8099/api/v1/ws/group_chat/{group_chat_id}`
- 持久化：本地文件存储在 `local_data/` 下
