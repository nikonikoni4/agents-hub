# 截图脚本运行说明

## 脚本列表

| 脚本 | 用途 |
|------|------|
| `scripts/screenshot_main.py` | 截图前端主界面（会话页、技能广场、角色管理） |
| `scripts/screenshot_create_dialog.py` | 截图新对话创建框（单聊/群聊模式） |

---

## 1. screenshot_main.py

截图前端主界面，保存到 `screenshots/main.png`。

## 运行方式

### 1. 前端服务已在运行时

```bash
python scripts/screenshot_main.py
```

### 2. 需要自动启动前端服务时

```bash
python skills/webapp-testing/scripts/with_server.py --server "cd frontend && npm run dev" --port 5173 -- python scripts/screenshot_main.py
```

### Windows PowerShell 注意事项

PowerShell 中路径需加引号，且不支持 `\` 续行：

```powershell
python "skills/webapp-testing/scripts/with_server.py" --server "cd frontend && npm run dev" --port 5173 -- python "scripts/screenshot_main.py"
```

---

## 2. screenshot_create_dialog.py

截图新对话创建框（自动点击"新建对话"按钮），保存到 `screenshots/create_dialog_*.png`（带时间戳）。

### 运行方式

**前端服务已在运行时：**

```bash
python scripts/screenshot_create_dialog.py
```

**自动启动前端服务：**

```bash
python skills/webapp-testing/scripts/with_server.py \
    --server "cd frontend && npm run dev" --port 5173 \
    -- python scripts/screenshot_create_dialog.py
```

**Windows PowerShell：**

```powershell
python "skills/webapp-testing/scripts/with_server.py" --server "cd frontend && npm run dev" --port 5173 -- python "scripts/screenshot_create_dialog.py"
```

### 输出说明

- `screenshots/create_dialog_YYYYMMDD_HHMMSS.png` — 单聊模式截图
- `screenshots/create_dialog_group_YYYYMMDD_HHMMSS.png` — 群聊模式截图（如果存在模式切换）

### 在循环中使用

适合在"截图→设计→实现→验证"的循环中使用，每次修改后运行获取最新截图：

```bash
# 1. 修改前端代码
# 2. 运行截图
python scripts/screenshot_create_dialog.py
# 3. 用 mimo-image MCP 查看截图
# 4. 根据反馈继续调整
```
