---

## 一、设计风格概述

**整体风格**：极简扁平化（Flat Design），类似 Cursor/Windsurf 的生产力工具风格。

- 无拟物元素、无毛玻璃质感
- 完全靠色彩层级和间距区分模块
- 以"功能性"为核心，视觉元素极度克制
- 氛围偏向专业、冷静的开发者工具风格

## 二、颜色方案 ⭐⭐⭐⭐⭐

### 核心色板（浅色主题）

| 角色 | 色值 | 说明 |
|------|------|------|
| **主内容区背景** | `#FAFAFA` | 极浅灰白，接近纯白的柔化背景 |
| **侧边栏背景** | `#F2F2F2` | 略深的浅灰，与主内容区形成轻量层级区分 |
| **卡片/输入框背景** | `#FFFFFF` | 纯白，靠背景色深浅差异区分 |
| **强调色** | `#F57C00` | 暖橙色，用于核心操作按钮、提示图标 |

### 文字色层级

| 层级 | 色值 | 用途 |
|------|------|------|
| 核心标题 | `#1A1A1A` | 深黑灰，高对比度，视觉焦点 |
| 正文/导航项 | `#333333` | 中深灰，核心操作文字 |
| 次要说明 | `#666666` | 浅灰，时间、快捷键提示 |
| 占位符 | `#999999` | 更浅灰，输入框占位文字 |
| 图标/次要按钮 | `#555555` | 深灰，与文字色系统统一 |

### 与当前 DESIGN.md 的差异

| 维度 | 当前 DESIGN.md | 参考图 |
|------|---------------|--------|
| 侧边栏背景 | `rgb(246, 246, 246)` = `#F6F6F6` | `#F2F2F2`（略深） |
| 主内容区背景 | `rgb(255, 255, 255)` = `#FFFFFF` | `#FAFAFA`（非纯白，更柔和） |
| 强调色 | `rgb(74, 158, 255)` = 蓝色 | `#F57C00` 暖橙色 |
| 文字主色 | `rgb(30, 30, 30)` | `#1A1A1A`（接近） |
| 文字次要色 | `rgb(100, 100, 100)` | `#666666`（更浅） |

**关键调整**：
1. 主内容区背景从纯白改为 `#FAFAFA`，减少刺眼感
2. 强调色从蓝色改为暖橙色 `#F57C00`
3. 次要文字色从 `#646464` 调整为 `#666666`

## 三、间距与布局 ⭐⭐⭐⭐

### 布局结构

```
┌─────────────────────────────────────────────────┐
│                   Top Bar (40px)                 │
├──────────┬────────────────────────┬─────────────┤
│          │                        │             │
│  左侧栏   │      主内容区 (flex:1)    │   右侧栏    │
│  ~15%    │        ~70%            │    ~15%     │
│ 200-220px│                        │  200-220px  │
│          │                        │             │
└──────────┴────────────────────────┴─────────────┘
```

### 关键间距

| 区域 | 间距值 | 说明 |
|------|--------|------|
| 标题与顶部边缘 | `60px` | 营造开阔操作感 |
| 标题与输入框之间 | `40-50px` | 视觉呼吸空间 |
| 侧边栏导航项垂直间距 | `16px` | 层级清晰 |
| 分组标题与子项间距 | `8px` | 紧凑但不拥挤 |
| 功能卡片间距 | `12-16px` | 适中 |
| 卡片内元素间距 | `6-8px` | 紧凑 |

### 与当前 DESIGN.md 的差异

| 维度 | 当前 DESIGN.md | 参考图 |
|------|---------------|--------|
| 左侧栏宽度 | `280px` | `200-220px`（更窄） |
| 右侧栏宽度 | `320px` | `200-220px`（更窄） |
| 主内容区留白 | `24px` | 更充足（40-60px） |

**关键调整**：
1. 左右侧栏从 280px/320px 收窄至约 220px
2. 主内容区增加更多留白

## 四、圆角与阴影 ⭐⭐⭐⭐

### 圆角规范

| 元素 | 圆角值 | 说明 |
|------|--------|------|
| 输入框 | `12px` | 大圆角，弱化边角生硬感 |
| 功能卡片/按钮 | `8-10px` | 中等圆角 |
| 小尺寸元素（图标容器） | `4px` | 极小圆角 |

### 阴影规范

| 元素 | 阴影 | 说明 |
|------|------|------|
| 输入框 | `0 2px 8px rgba(0,0,0,0.05)` | 极淡单层柔和外阴影 |
| 功能卡片 | **无阴影** | 靠背景色深浅区分层次 |
| 侧边栏 | **无阴影** | 扁平设计 |

### 与当前 DESIGN.md 的差异

| 维度 | 当前 DESIGN.md | 参考图 |
|------|---------------|--------|
| 输入框圆角 | `16px` | `12px` |
| 模块卡片圆角 | `8px` | `8-10px`（接近） |
| 右侧栏模块阴影 | `0 1px 3px` | 无阴影（扁平） |
| 主对话区左侧圆角 | `12px 0 0 12px` | 无此设计（平直边缘） |

**关键调整**：
1. 输入框圆角从 16px 减小到 12px
2. 移除卡片/模块阴影，改为纯背景色区分
3. 主对话区取消左侧圆角，改为平直边缘（与侧边栏平齐）

## 六、按钮与卡片样式 ⭐⭐⭐⭐

### 强调按钮（如"发送"）

```css
background: #F57C00;
color: #FFFFFF;
border-radius: 8px;
/* 或圆形发送按钮 */
border-radius: 50%;
background: #333333;
```

### 功能卡片（右侧栏模块）

```css
background: #F5F5F5;
border-radius: 8px;
padding: 12px;
/* 无阴影，无边框 */
```

### 侧边栏操作项

```css
/* 纯文字+图标，无背景 */
padding: 8px 12px;
color: #333333;
/* hover 时浅灰背景 */
background: rgba(0, 0, 0, 0.05);
```

## 七、设计规范总结（给前端开发者）

### 必须调整的 5 个核心点

1. **颜色调整**：
   - 主内容区背景：`#FFFFFF` → `#FAFAFA`
   - 侧边栏背景：`#F6F6F6` → `#F2F2F2`
   - 强调色：蓝色 → `#F57C00`（暖橙色）

2. **尺寸调整**：
   - 左侧栏宽度：`280px` → `220px`
   - 右侧栏宽度：`320px` → `220px`
   - 输入框圆角：`16px` → `12px`

3. **阴影移除**：
   - 移除右侧栏模块阴影
   - 仅保留输入框的极淡阴影

4. **布局调整**：
   - 主内容区增加留白（从 24px 到 40px+）
   - 主对话区考虑取消左侧圆角，改为平直边缘

5. **文字色调整**：
   - 次要文字：`#646464` → `#666666`
   - 占位符文字：`#999999`

### CSS 变量更新建议

```css
:root {
  /* 背景色 */
  --bg-sidebar: #F2F2F2;        /* 原 #F6F6F6 */
  --bg-main: #FAFAFA;           /* 原 #FFFFFF */
  --bg-right-module: #F5F5F5;   /* 新增 */
  --bg-input: #FFFFFF;          /* 保持 */

  /* 文字色 */
  --text-primary: #1A1A1A;      /* 原 #1E1E1E */
  --text-secondary: #333333;    /* 原 #646464，提升层级 */
  --text-tertiary: #666666;     /* 原 #969696 */
  --text-placeholder: #999999;  /* 新增 */

  /* 强调色 */
  --accent-color: #F57C00;      /* 原 #4A9EFF，改为暖橙 */

  /* 圆角 */
  --radius-input: 12px;         /* 原 16px */
  --radius-card: 8px;           /* 保持 */
  --radius-sm: 4px;             /* 保持 */

  /* 阴影 */
  --shadow-input: 0 2px 8px rgba(0, 0, 0, 0.05);
  --shadow-card: none;          /* 移除阴影 */
}
```
