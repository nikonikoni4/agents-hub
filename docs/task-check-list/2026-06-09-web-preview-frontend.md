# 网页预览卡片前端功能

## 任务背景

后端已完成 web_preview 字段全链路支持，前端需要实现对应的展示和交互。

## 任务目标

Agent 返回网页预览时，用户可以在消息中看到预览卡片，点击后在右侧栏查看网页内容。

## 检查清单

### 消息内卡片
- [ ] Agent 回复带 web_preview 时，消息下方显示预览卡片
- [ ] 卡片展示网页标题和来源地址
- [ ] 点击卡片在右侧栏打开网页预览
- [ ] 已打开的卡片有视觉反馈（激活态）

### 右侧栏预览
- [ ] 右侧栏新增"网页"tab，独立于现有"预览"tab
- [ ] tab 内通过 iframe 展示网页内容
- [ ] 无预览时显示空态提示
- [ ] 提供"在浏览器中打开"的入口
