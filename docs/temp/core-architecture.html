依赖规则

  • foundation — 零依赖，提供枚举、数据模型、异常、渲染、路径管理
  • context — 依赖 foundation，管理会话、消息持久化、上下文加载
  • communication — 依赖 foundation，管理消息路由、Agent 调用、任务
  • agent — 依赖 foundation + context + communication，Agent 执行逻辑
  • orchestration — 依赖所有下层，协调组件生命周期

AgentCall 生命周期
