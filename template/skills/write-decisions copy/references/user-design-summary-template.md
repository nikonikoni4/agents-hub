# User Design Summary 模板

如果 `user-design-summary.md` 不存在，创建时使用此模板；如果已存在，在对应章节追加观察。

```markdown
# User Design Summary

## 目的

记录用户在重大设计决策中的偏好、判断方式和风险倾向，为后续 agent 自主决策提供参考，也为用户复盘决策方式提供材料。

## 决策行为观察

| 日期 | 关联决策 | 观察 | 可参考性 | 风险提醒 |
| ---- | -------- | ---- | -------- | -------- |
| YYYY-MM-DD | [决策标题](./YYYY-MM-DD-{具体内容}.md) | 用户在本次决策中体现出的判断方式 | 后续 agent 可参考的条件 | 不应盲目复用的边界或风险 |

## 决策偏好总结

- 当前可观察到的稳定偏好。
- 偏好成立的上下文条件。
- 可能导致误判的倾向。

## 给后续 agent 的参考

- 可以参考的判断原则。
- 必须重新确认的前提。
- 不建议直接沿用的做法。
```
