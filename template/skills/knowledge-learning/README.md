# Knowledge Learning Skill

一个基于评估驱动的渐进式知识学习系统技能。

## 核心特性

- **评估驱动教学** - 先了解用户现有知识水平
- **自适应内容** - 根据用户理解程度调整教学内容
- **渐进式学习** - 从基础开始，逐步深入
- **互动问答** - 用户主导的学习探索
- **学习报告** - 可选生成学习历程和知识报告

## 使用场景

当用户说以下内容时触发此技能：
- "我想学习X"
- "教我X"
- "帮我理解X概念"
- "学习X知识"
- "explain X to me"

## 学习报告生成

当用户请求总结时，生成两个文件：

1. **学习历程文档** (`learning-journey-[主题]-[日期].md`)
   - 记录完整的学习过程
   - 包含评估问题和用户回答
   - 分析理解障碍点
   - 总结学习进度

2. **知识报告文档** (`knowledge-report-[主题]-[日期].md`)
   - 概念定义和核心要点
   - 详细阐述和常见误解
   - 进阶方向和参考资源
